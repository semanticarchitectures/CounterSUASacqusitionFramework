# 02-Requirements: Modular Asset Protection

This directory contains the requirement taxonomies and specific constraint sets for various asset types.

## Asset Categories
1. **Fixed High-Value Sites**: Airfields, power plants, government buildings.
2. **Extended Sites**: Cities, sprawling industrial complexes.
3. **Floating/Maritime**: Ships, naval groups, offshore platforms.
4. **Mobile Assets**: Ground convoys, VIP transport.
5. **Personnel**: Point protection for individuals or small teams.

## Requirement Modules
- **Detection**: Range, sensitivity, sensor fusion requirements.
- **Identification**: Classification confidence, automated vs. manual ID.
- **Alerting**: Latency, UI/UX for operators, false alarm thresholds.
- **Mitigation**: Kinetic vs. non-kinetic effectors, self-protection rules.
- **C2 Integration**: Interoperability with existing defense networks.
